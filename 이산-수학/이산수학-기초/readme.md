### 1강 이산수학 개요
- 불연속적인 숫자를 다루는 수학

### 3강 역, 이, 대우
- 진리표: 각 명제 사이의 관계식의 진릿값을 보여주는 표   
![image](https://user-images.githubusercontent.com/28378553/124711977-b18efc80-df39-11eb-80ce-8828a25e19de.png)
- 본 명제: p->q
- 역: q->p
- 이: ~p->~q
- 대우: ~q->~p
